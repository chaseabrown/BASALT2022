Definitions

  • Description is what the item is called and (Minecraft ID Name) is the string value that is used in game commands.
  • Minecraft ID is the Internal number for the item.

Example of how to use a Minecraft Name

You will need to know the name of an item when using the /give command.

For example, you use the honey_block name in the /give command when you want to give the player called DigMinecraft 64 honey blocks.

/give DigMinecraft honey_block 64
